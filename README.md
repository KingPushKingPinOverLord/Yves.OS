Yves.OS

A sleek, interactive web-based operating system simulation built entirely with vanilla HTML, CSS, and JavaScript. Yves.OS features a modern glassmorphism interface, dynamic canvas backgrounds, and a suite of fully functional productivity and entertainment apps.

✨ Features

Window Management: Draggable, maximizable, and closable app windows.

Dynamic UI: Smooth glassmorphism panels, an interactive macOS-style dock, and an animated HTML5 Canvas blob background.

Integrated Applications:

Terminal: A functional CLI (type help to explore commands).

Workspace: A Notion-style rich-text document editor.

Tasks: A project and to-do list tracker with live progress bars.

Focus Timer: Built-in Pomodoro timer for productivity.

Settings: Live UI customization (adjust blur, saturation, and aura colors).

Break Time: A fully playable Flappy Bird clone using Canvas-like DOM manipulation.

Easter Eggs: Use the Terminal to uncover hidden ASCII art backgrounds (try typing matrix, lucy, or david).

🚀 How to Run

This project is completely self-contained and requires no build tools, frameworks, or dependencies.

Download or clone the repository.

Open the index.html file directly in any modern web browser (Chrome, Firefox, Safari, Edge).

🛠️ Tech Stack

HTML5: Semantic structure and HTML5 Canvas.

CSS3: Native CSS variables, flexbox, grid, animations, and backdrop-filter for glassmorphism.

Vanilla JavaScript (ES6+): DOM manipulation, window state management, and physics loops via requestAnimationFrame.
